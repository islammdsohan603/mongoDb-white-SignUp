## ✨ Features

### 🔓 Public Pages

- **Home Page** — Hero section, feature cards, authentication CTA
- **Sign Up** — User registration with validation & MongoDB save
- **Sign In** — Login with password toggle & session creation

### 🔒 Protected Dashboard

- **User Welcome Section** — Personalized greeting with date
- **Statistics Cards** — Total Sessions, Login Count, Account Status, Security
  Level
- **Activity Feed** — Real-time login & activity history
- **User Profile Card** — User details, role, and auth provider info
- **Sidebar Navigation** — Dashboard, Profile, Analytics, Settings, Logout
- **Session-based Navbar** — Shows different UI for logged-in/logged-out users
- **Dark Mode UI** — Sleek dark theme throughout
- **Fully Responsive** — Works on all screen sizes
